# Support

## Where to Ask for Help

- Usage questions, setup problems, and feature discussions:
  - Open a GitHub Discussion (if enabled), or
  - Open a GitHub issue using the relevant template.

## Bug Reports

Use the Bug Report template and include:
- environment details,
- reproduction steps,
- logs/errors,
- expected vs actual behavior.

## Security Issues

Do not report security issues publicly.  
See `SECURITY.md` for private disclosure instructions.

## Documentation Issues

For docs corrections or improvements, open a docs issue or PR directly.

## SLA

This open-source repository is maintained on a best-effort basis.  
No guaranteed response time is implied unless explicitly contracted.
